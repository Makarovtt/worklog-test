## Возможности

- Вход и регистрация (JWT)
- Журнал работ: таблица, фильтры (период, вид работ, исполнитель), сортировка и пагинация
- Создание и редактирование записи
- Массовое добавление записей (одна транзакция на бэкенде)
- Массовое удаление и смена даты у выбранных строк
- Удаление с кнопкой «Отменить» (soft delete + восстановление через API)

## Стек

| Слой     | Технологии                                                                     |
| -------- | ------------------------------------------------------------------------------ |
| Frontend | React 18, TypeScript, Vite, Ant Design 5, TanStack Router, TanStack Query, FSD |
| Backend  | NestJS 10, Prisma, PostgreSQL 16, JWT, Swagger                                 |
| Инфра    | Docker, docker-compose, GitHub Actions                                         |

## Почему

Ant Design — готовые таблицы, формы, фильтры и модалки; подходит для рабочих интерфейсов, не нужно собирать UI с нуля.
TanStack Query — кеширование запросов API — повторно не дергает API, если данные ещё актуальны, автоматическое обновление после CRUD
TanStack Router — типобезопасная маршрутизация; фильтры и пагинация в URL, можно делиться ссылкой на нужный вид журнала.

NestJS + Prisma + PostgreSQL — один TypeScript на фронте и бэке, переиспользование типов - удобно; Nest даёт структуру API, Prisma — типы и миграции, PostgreSQL — надёжное хранение.

## Быстрый старт (Docker)

Требования: Docker Desktop (или Docker Engine + Compose v2).

```bash
docker compose up --build
```
